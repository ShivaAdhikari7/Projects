The basic language of the web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All website and web applications are made using three fundamental technologies: HTML, CSS and JavaScript.This are the language of the web.

In this post, let's focus on the HTML. We will learn what HTML is all about, and why you should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It's the Markup language that developers use to structure and describe the content of the page(not a programming language).

HTML consists of the elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To able to use the fundamental web dev language.
  • To handcraft beautiful websites instead of relying on tools like Wordpress or Wix.
  • To build web applications.
  • To impress friends
  • To have fun.
